Considerando o código abaixo:

<'p class="paragrafo">

p {
    color: blue;
}

p.paragrafo {
    color: red;
}

.paragrafo {
    color: purple;
}
Qual será a cor final do meu parágrafo?

Resposta:


red


Alternativa correta! O seletor p.paragrafo é o mais forte, e a cor aplicada é a vermelha (red).
